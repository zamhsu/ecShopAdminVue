# ecShopAdminVue
一個由Vue 3所建立的的簡易電商網站（後台管理端）。\
[Demo](https://agreeable-sky-00319b600.1.azurestaticapps.net/)

## 主要功能

- [ ] 管理後台帳號
- [x] 管理商品
- [x] 查看訂單
- [x] 管理優惠券
- [ ] 管理系統參數

## 使用技術
* Vue 3
* TypeScript
* Axios
* Bootstrap

## 相關的專案
